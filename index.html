cd Programming/LiU/TDP001


För att initiera projektet skrev jag följande:

git init


Sedan skapade jag en .gitignore och sparade de nuvarande filerna inför committen med:

git add .


Efter det skapade jag en commit med:

git commit -m "add labb1"


Sedan lade jag till mitt GitLab-projekt som remote-repo med:

git remote add origin https://gitlab.liu.se/vadno575/tdp001


Och till sist, för att få min commit att synas på servern, skrev jag:

git push -u origin

Deluppgift 0

För att hämta "resultatet.txt" använde jag curl för att hämta filen från servern till lokal fil - "tempRes.txt" och iconv för att konvertera den till UTF-8 och skriva till "resultat.txt":

curl https://www.ida.liu.se/~TDP001/2025/material/resultat.txt -o tempRes.txt | iconv -f ISO-8859-1 -t UTF-8 tempRes.txt > resultat.txt


Deluppgift 1

För att hitta alla rader med namn som avslutas med "on" använde jag grep med regex 'on\b' där \b är en "word boundary":

grep 'on\b' resultat.txt


Deluppgift 2

För att skriva ut alla efternamn utom de som innehåller "son" använde jag cut för att få tag på alla efternamn och grep på dessa efternamn med -v som är flagga för "invert match":

cut -d' ' -f2 resultat.txt | grep -v son | head -n5


Deluppgift 3

För att hitta antalet namn med "an" använde jag cut för att dra ut alla namn, grep på dessa namn med "an" och wc -l för att räkna antalet rader (-l flagga för att räkna antalet rader istället för antalet ord/tecken):

cut -d' ' -f1-2 resultat.txt | grep an | wc -l

Uppdaterad: 2025-09-12

Deluppgift 1

För att hitta alla python filer använde jag -name "*.py"; För att hitta alla +x filer använde jag -executable; för att kontrolera att de har #! använde jag -exec med grep kommando som kontrallerade att filar som matchade än så länge (markerad med {}) innehåller också '#!/usr/bin/python3' och efteråt printade ut filnamn med -l

find ./ -name "*.py" -executable -exec grep -l '#!/usr/bin/python3' {} \;


Deluppgift 2

För att hitta alla filer som har storleken mer än 5MB i home katalogen använde jag find ~ (home katalogen) och -size +5M som filtrerar alla filer med storleken över 5M

find ~ -size +5M

C-e M-b M-b M-b M-b M-f
