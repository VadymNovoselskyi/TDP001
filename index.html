Uppg1

Uppdaterad: 2025-09-02

Jag loggade in på GitLab, lade till min SSH-nyckel och skapade ett tomt repository med namnet TDP001 via GUI.


Efter det öppnade jag terminalen och navigerade till projektkatalogen med:

cd Programming/LiU/TDP001


För att initiera projektet skrev jag följande:

git init


Sedan skapade jag en .gitignore och sparade de nuvarande filerna inför committen med:

git add .


Efter det skapade jag en commit med:

git commit -m "add labb1"


Sedan lade jag till mitt GitLab-projekt som remote-repo med:

git remote add origin https://gitlab.liu.se/vadno575/tdp001


Och till sist, för att få min commit att synas på servern, skrev jag:

git push -u origin
